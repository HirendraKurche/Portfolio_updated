## ğŸš€ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for development)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/HirendraKurche/Portfolio_updated.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Portfolio_updated
   ```

3. **Open the project**
   - Simply open `index.html` in your web browser
   - Or use a local server (recommended):
   
   **Using Python:**
   ```bash
   python -m http.server 8000
   ```
   
   **Using Node.js (http-server):**
   ```bash
   npx http-server
   ```
   
   **Using VS Code Live Server:**
   - Install the Live Server extension
   - Right-click on `index.html` and select "Open with Live Server"

4. **View the website**
   - Navigate to `http://localhost:8000` (or the port specified by your server)

## ğŸ¨ Customization

### Changing Content
1. **Personal Information**: Edit the text in `index.html`, `about.html`
2. **Projects**: Update project details in `work.html` and individual project pages
3. **Images**: Replace images in `assets/img/` with your own
4. **Colors**: Modify CSS variables in `assets/css/style-new.css`

### Key Sections to Update
- Profile image: `assets/img/A Guy With Shirt.png`
- Project images: `assets/img/SelectedProjects/`
- Contact information: Update email and phone in `index.html` and `contact.html`
- Social media links: Update GitHub, LinkedIn, Behance URLs
